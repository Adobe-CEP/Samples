Illustrator Host Adapter Demo Extension

This sample extension adds a 'new document' event listener and displays an alert dialog upon receiving the event.

  • See the AIEvent class defined in /js/host_adapter/ai_host_adapter.js for full list of event types.
  • See /js/main.js to see how event listeners are added.
  • Ensure you have copied the host adapter plug-in(s) to the plug-ins folder inside the installation footprint of Illustrator. (e.g. on Mac, copy plug-ins/mac/AIHostAdapter.aip to /Applications/Adobe Illustrator CC 2014/Plug-ins.localized/AIHostAdapter.aip)
